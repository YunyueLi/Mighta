# Screenshots

Drop your screenshots in here. The main `README.md` expects these filenames:

```
docs/
├── hero-dark.png        # landing hero, dark mode (1600×900 ideal)
├── hero-light.png       # landing hero, light mode
├── landing-dark.png     # full landing scrolled, dark
├── landing-light.png    # full landing scrolled, light
├── timeline.png         # spawn results, timeline view
├── cards.png            # spawn results, cards view
├── importer.png         # importer modal with sample text
└── settings.png         # settings page, providers visible
```

Take them with macOS `cmd+shift+4` (region) or use `mcp__Claude_Preview__preview_screenshot` if you have Claude Code wired up.

Compress with `pngquant` or `tinypng` before committing.
